# 🎮 The Runes of Statera

A story-driven 2D RPG built with **HTML, CSS, and JavaScript**.

Play as a lone warrior tasked with collecting the ancient **Runes of Statera**—but as the journey unfolds, you’ll discover that not everything is as it seems...

---

## 🌐 Play the Game

👉 https://matthiaseaton.github.io/runes-of-statera-game/

---

## 🧩 Features

* ⚔️ Real-time combat system
* 🧠 Enemy AI with attack states and animations
* 🎒 Inventory system (weapons, potions)
* 💬 Dialogue system with typewriter effect
* 🌲 Multiple scenes (Forest, Castle, Throne Room, etc.)
* 🔮 Rune collection and progression system
* 🧙 Branching story choice (multiple paths & endings)
* 🎵 Background music and sound effects
* 🎬 Fade transitions and cinematic text sequences

---

## 🛠️ Built With

* HTML5 Canvas
* Vanilla JavaScript (no frameworks)
* CSS
* Custom game loop & state management

---

## 🚀 Getting Started (Local)

1. Clone the repo:

```bash
git clone https://github.com/MatthiasEaton/runes-of-statera-game.git
```

2. Open the project folder

3. Open `index.html` in your browser
